# Microsoft Sentinel Data Connectors

## Overview

Microsoft Sentinel was configured as the cloud SIEM layer for cross-source telemetry correlation.

Data connectors enable centralized threat hunting using KQL.

## Data Sources Connected

### Microsoft Defender for Endpoint

Provides:

- Alerts
- Device timelines
- Behavioral detections

### Windows Security Events

Provides:

- Authentication activity
- Account changes
- Privilege modifications

### Identity Telemetry

Provides:

- Sign-in activity
- Account anomalies

## Detection Value

Sentinel enables:

- Cross-source correlation
- Incident timeline reconstruction
- Cloud-based threat hunting

## Validation

Confirmed:

- Data connectors active
- Events searchable via KQL
- Incidents generated during lab simulations
