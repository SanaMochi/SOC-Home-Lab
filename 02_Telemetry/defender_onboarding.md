# Microsoft Defender for Endpoint Deployment

## Overview

Endpoints in the lab environment were onboarded into Microsoft Defender for Endpoint (MDE) to provide endpoint detection and response (EDR) telemetry.

MDE provides behavioral detection capabilities beyond traditional log analysis by monitoring system activity in real time.

Endpoints onboarded:
- DC01 — Active Directory Domain Controller
- CLIENT01 — Domain-joined workstation

## Telemetry Provided

Microsoft Defender for Endpoint collects enriched behavioral telemetry including:
- Process execution
- Command-line arguments
- Parent-child process relationships
- Network connections
- File and registry modifications
- Security alerts generated by Microsoft detection models

This telemetry enables detailed investigation of suspicious activity occurring on endpoints.

## Detection Value

Defender telemetry supports detection of behaviors such as:
- Malicious PowerShell execution
- Credential dumping attempts
- Suspicious process injection
- Lateral movement techniques
- Post-exploitation activity

These detections supplement log-based detection rules developed within Splunk and Sentinel.
